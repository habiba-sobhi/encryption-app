# Encryption & Decryption Application

## 📌 Description
A Python-based GUI application that implements encryption and decryption using classical cryptography techniques.

## 🚀 Features
- Caesar Cipher
- Vigenère Cipher
- Multi-layer encryption
- User-friendly GUI

## 🛠️ Tech Stack
Python, Tkinter

## ▶️ How to Run
1. Run the Python file
2. Use GUI to encrypt/decrypt text
